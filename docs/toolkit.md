# Toolkit Reference

Overview of skills, agents, and hooks generated by `parallax init`.

## Skills

Invoked as `/skill-name` in Claude Code.

| Skill | Description |
|-------|-------------|
| `/hypothesis` | Formalize hypothesis with alternatives and success/failure criteria before implementation. |
| `/experiment` | Scaffold structured experiment manifest linked to a hypothesis. |
| `/session-start` | Bootstrap session: review handoffs, memory, project state. |
| `/handoff` | Generate structured session summary for context continuity. |
| `/audit` | Systematic project health check: code, tests, docs, deps, architecture, rigor. |
| `/doc-sync` | Audit all living docs against codebase, report inconsistencies, offer fixes. |
| `/test-integrity` | Review branch diff for test weakening patterns. Last line of defense before commit. |
| `/grill-me` | Stress-test a plan or design through relentless decision-tree interrogation. |
| `/manuscript-review` | Peer review critique of LaTeX papers or Beamer presentations (read-only). |
| `/latex-guide` | LaTeX/BibTeX troubleshooting reference: error patterns, fixes, Beamer tips. |

## Agents

Launched via `@agent-name` or from agent definitions in `.claude/agents/`.

| Agent | Description |
|-------|-------------|
| `hypothesis-explorer` | Investigate a hypothesis in a worktree -- search code, tests, data for evidence. |
| `experiment-runner` | Design and execute experiments: create test scripts, run them, record results. |
| `literature-reviewer` | Search and summarize relevant papers, references, and prior work. |
| `result-validator` | Validate scientific rigor: reproducibility, statistics, methodology (read-only). |
| `paper-writer` | Draft and iterate scientific papers in LaTeX with compilation checks. |
| `presentation-writer` | Create Beamer presentations from existing results and papers. |
| `manuscript-reviewer` | Review LaTeX papers or Beamer presentations with constructive critique. |

## Hooks

Automatic enforcement scripts configured in `.claude/settings.json`.

| Hook | Trigger | Behavior |
|------|---------|----------|
| `test_guard.py` | PreToolUse (Edit/Write) | **Blocks** edits that weaken tests (skip markers, trivial assertions, noqa). |
| `lint_check.py` | PostToolUse (Edit/Write) | Runs ruff on edited Python files, reports issues (informational). |
| `stop_check.py` | Stop | Reminds about uncommitted changes and suggests `/handoff` (informational). |

## Token Tiers

Control model selection per agent via `parallax init --token-tier` or `parallax config set token-tier`.

| Tier | Strategy |
|------|----------|
| **pro** (default) | Conservative: haiku exploration, sonnet validation |
| **5x** | Balanced: opus exploration, sonnet runner |
| **20x** | Generous: opus for most tasks |
| **api** | Unconstrained: opus everywhere |
